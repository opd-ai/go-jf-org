# CLI Commands

This directory will contain CLI command implementations using Cobra.

## Planned Commands

- `root.go` - Root command and application setup
- `scan.go` - Scan directories for media files
- `organize.go` - Organize files into Jellyfin structure
- `preview.go` - Preview changes without executing
- `verify.go` - Verify organized structure
- `config.go` - Configuration management
- `rollback.go` - Rollback operations

See [IMPLEMENTATION_PLAN.md](../IMPLEMENTATION_PLAN.md) for details.
